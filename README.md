# Certificados — Portfólio de Formação em Cibersegurança

Repositório que reúne certificados e comprovantes de conclusão de cursos na área de **segurança da informação**, **pentest**, **ethical hacking** e temas relacionados.

---

## Sobre

Este projeto funciona como um portfólio centralizado dos certificados obtidos em formações em cibersegurança, permitindo consulta e compartilhamento de forma organizada.

---

## Conteúdo do repositório

Os certificados estão organizados em pastas por tema. Resumo por área:

| Pasta | Conteúdo |
|-------|----------|
| **Fundamentos** | Introdução ao Hacking e Pentest 2.0, Fundamentos de Segurança da Informação, Redes e Criptografia para Pentesters, certificados gerais de introdução |
| **Pentest** | Metodologias de Pentest na Prática, Pentest em Infraestrutura, Recon, Pós-exploração Linux, Experiência Riachuelo Cibersegurança |
| **Web** | Ataques em Aplicações Web, Desenvolvimento Web para Pentesters, Inteligência Artificial para Pentest Web |
| **Redes e infra** | Ataques em Redes Wi-Fi, Denial of Service (DoS) e Botnets |
| **Bug Bounty** | Automação e estratégias em Bug Bounty |
| **Mobile** | Desenvolvimento de Malwares para Android |
| **Programação** | Fundamentos de C para Pentesters, Introdução ao Python e Algoritmos, Python Orientado a Objetos |
| **Sistemas** | Linux e Shell |
| **Engenharia social** | Técnicas de Phishing e Engenharia Social |
| **IA** | A Era da IA (Machine Learning, LLMs, IA Generativa e Agentes), Fundamentos de Modelos de Linguagem de Grande Escala, Introdução à Engenharia de Prompts, Técnicas de Engenharia de Prompt |
| **Outras plataformas** | Introduction to Cybersecurity, Ethical Hacker, Cyber Threat Management |
| **Outros** | Perfil/currículo (Samuel_Ziger.pdf) e demais documentos |

---

## Estrutura de pastas

```
Certificado/
├── Fundamentos/       # Introdução, redes, criptografia, segurança da informação
├── Pentest/          # Metodologias, recon, infraestrutura, pós-exploração, Riachuelo
├── Web/              # Ataques web, dev web para pentesters, IA para pentest web
├── Redes e infra/    # Wi-Fi, DoS e botnets
├── Bug Bounty/       # Automação e estratégias em bug bounty
├── Mobile/           # Malwares para Android
├── Programação/      # C, Python (introdução e OO)
├── Sistemas/         # Linux e Shell
├── Engenharia social/# Phishing e engenharia social
├── IA/               # LLMs, IA generativa, engenharia de prompts
├── Outras plataformas/ # Certificados externos (Cisco/NetAcad etc.)
└── Outros/           # Perfil e demais documentos
```

## Uso

- **Consulta:** abra os PDFs diretamente para visualizar os certificados.
- **Compartilhamento:** use os arquivos ou links deste repositório para demonstrar suas qualificações em processos seletivos ou redes profissionais.
- **Backup:** o repositório Git serve também como cópia versionada dos certificados.

---

## Observações

- **Uso exclusivo:** este repositório é um portfólio pessoal. Os certificados aqui são **meus**; não há propósito de outras pessoas adicionarem certificados — o uso é exclusivamente meu.
- Os certificados são emitidos pelas respectivas instituições/plataformas (ex.: Solyd e demais provedores dos cursos).
